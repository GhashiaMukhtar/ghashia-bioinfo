# Transcriptomic Analysis of Skeletal Muscle in Type 2 Diabetes Mellitus  
**GSE25724 • November 2025**

### Dataset
- GEO: [GSE25724](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE25724)  
- Samples: 15 T2D + 15 healthy controls

### Methods
GEO2R → DAVID (KEGG enrichment)

### Key Findings
Strong enrichment in hypertrophic & dilated cardiomyopathy pathways in diabetic muscle tissue.
